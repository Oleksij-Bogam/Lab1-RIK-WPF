# Lab1 — Survey Application (WPF)

## Опис проєкту

Цей застосунок реалізує **форму опитування користувача**, створену з використанням технології **WPF**.  
Програма дозволяє вводити відповіді на запитання через графічний інтерфейс та зберігати результати у файл на диску.

Метою лабораторної роботи є ознайомлення з основами створення **GUI-застосунків у WPF** та роботою з елементами інтерфейсу.

## Функціональність

Застосунок забезпечує:

- відображення форми опитування
- введення відповідей користувача через текстові поля
- обробку подій натискання кнопок
- збереження введених даних у файл
- роботу з елементами WPF-інтерфейсу

## Використані технології

- **C#**
- **.NET**
- **WPF (Windows Presentation Foundation)**
- **XAML**
- **Visual Studio**

## Приклад роботи програми

1. Користувач запускає застосунок.
2. На екрані з'являється форма з питаннями.
3. Користувач вводить відповіді у текстові поля.
4. Після натискання кнопки дані зберігаються у файл.

## Результат

Після завершення опитування створюється файл, що містить відповіді користувача.

- створення графічного інтерфейсу за допомогою **XAML**
- обробка подій користувача
